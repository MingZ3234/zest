Zest is an easy-to-use unit testing tool based on the Spring framework. It separates test data and test code, and automatically verifies the results of RMDB (SQL) and MongoDB

Supported Versions
------------------

- Support for 
    - Java 8
    - Spring 4 plus
    - JUnit 4 plus
